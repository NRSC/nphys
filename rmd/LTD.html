The raw data is nested within ABF, but can be easily accessed with the dfs_ABF() function, by feeding field$ABF into the function. For more on pulling data from a nnest, see the vignette on the nnest.


dfs = dfs_ABF(field$ABF)

Once we have our data, we want to observe and analyze it. To do this, we can pass the data to predefined functions to asses it. It is not necessary to always isolate the data using the dfs_ABF() function, as it is fully integrated into other functions to automatically draw the data. Therefore, we can use functions such as pullSweeps() to directly pull relevant traces and avgSweeps() to format them for viewing.

traces = pullSweeps(field$ABF, pull = "PreC-Bl")
